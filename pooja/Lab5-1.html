Concept of WWW

  • it stands for world wide web.
  • it is basically a system of internet servers that support specially formatted documents.
  • it is collection of text pages, videos, digital photographs and animations we can access over the internet.
  • users access the world wide web facilities via client called a web browser.

HTTP Protocol

  1. HTTP stands for Hypertext Transfer Protocol.
  2. HTTP is based on the client-server architecture model and a stateles request/response protocol
  3. 404 file not found is common http status code.
